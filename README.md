最近迷上了 Forza Horizon 4，但因為 Steam 版本缺乏控制器震動的體驗，我特地開發了一套自訂的板機震動設定，增強了遊戲的臨場感，包括：
- **ABS 震動提醒**：當觸發 ABS 時，板機會隨之發生震動，讓你更了解煞車狀態
